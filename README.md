# 🚀 Employee Attrition Prediction System  

## 🔍 Data-Driven Workforce Analytics with Machine Learning, Visualization & Security  

---

## 📌 Overview  
Employee attrition is a major challenge for organizations, leading to increased hiring costs, productivity loss, and workforce instability.  

This project builds a **complete end-to-end Machine Learning pipeline** to predict employee attrition and provide actionable insights.  

💡 Unlike basic ML projects, this system integrates:  
- 🏆 Leaderboard system (model comparison like Kaggle)  
- 📊 Advanced visualizations (ROC, Confusion Matrix, Feature Importance)  
- 🔐 Encryption module (secure storage of results)  

---

## 🎯 Objectives  
- Predict whether an employee is likely to leave  
- Compare multiple ML models and identify the best performer  
- Visualize model performance for interpretability  
- Secure sensitive outputs using encryption  

---

## 🧠 Machine Learning Models  
- Logistic Regression  
- Decision Tree  
- Random Forest  
- XGBoost (with Hyperparameter Tuning)  

---

## ⚙️ ML Pipeline  
1. Data Loading & Exploration  
2. Data Preprocessing & Encoding  
3. Feature Engineering  
4. Model Training  
5. Model Evaluation  
6. Leaderboard Generation  
7. Encryption of Results  

---

## 🏆 Leaderboard System  
A custom-built leaderboard that ranks models based on:  
- Accuracy  
- Precision  
- Recall  
- F1-Score  

---

## 📊 Visualizations  
- Confusion Matrix  
- ROC Curve  
- Feature Importance  
- Probability Distribution  
- Correlation Heatmap  

---

## 🔐 Encryption Module  
- Uses Fernet Encryption  
- Encrypts leaderboard results  
- Supports decryption  

---

## 📈 Output  
- Model comparison  
- Leaderboard  
- Encrypted results  
- Visual plots
