# How to Submit Your Score to Leaderboard

## Step 1: Install GitHub CLI (one time only)

1. Download GitHub CLI from: https://cli.github.com/
2. Run the installer (just click Next, Next, Finish)
3. Restart your computer

## Step 2: Login to GitHub

Open **Command Prompt** (search for "cmd" in Windows) and type:

```bash
gh auth login
